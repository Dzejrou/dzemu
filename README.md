#DZEMU


`dzemu` is an emulation/compilation library. Its goal
is to offer emulators for various architectures and assemblers/compilers
targeting these architectures.

## Current state:
 * Emulators:
  * MCS6502
 * Assemblers:
  * MCS6502
 * Disassemblers:
  * MCS6502
 * Compilers
  * An extensible lexer
